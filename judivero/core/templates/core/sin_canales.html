No hay canales configurados

Para comenzar a usar el sistema de moderación, primero debes agregar los canales de Twitch donde eres moderador.

Agregar mi primer canal Ir al panel de administración

¿Cómo agregar un canal?

  1. Haz clic en "Agregar mi primer canal"
  2. Completa los datos del canal:
    • • Nombre: Nombre del canal (ej: "ElRubius")
    • • Streamer: Nombre de usuario del streamer (ej: "elrubius")
    • • URL: Link del canal de Twitch (opcional)
    • • Descripción: Notas sobre reglas o configuración (opcional)
  3. Guarda y comienza a gestionar comandos, notas y baneos

Método alternativo: Usar Django Shell

Si prefieres agregar canales mediante código, puedes usar el shell de Django: 

python manage.py shell

from core.models import CanalTwitch

canal = CanalTwitch.objects.create(
    nombre="NombreDelCanal",
    streamer="usuario_twitch",
    descripcion="Notas sobre el canal",
    url_twitch="https://twitch.tv/usuario",
    color_distintivo="#9147ff"
)

print(f"✓ Canal creado: {canal.nombre}")
